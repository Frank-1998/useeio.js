We ran exiobase_downloads.py which combines US BEA and EXIOBASE data emissions factors for annual trade data reporting. Exiobase provides the equivalent to M, N, and x which is used in USEEIO models for import emissions factors. Exiobase also provides gross trade data which has no equivalent in USEEIO.
